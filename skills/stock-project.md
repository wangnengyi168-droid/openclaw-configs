# SKILL.md — stock-project

## 描述
老王股票项目 v2.0（极简版）—— ETF 投资分析和评分系统。

**注意**：本技能的实际代码和数据在 `~/.openclaw/workspace/stock-project/` 目录下，不在 skills 目录内。

核心功能：
- **持仓管理**：跟踪 ETF 持仓、买入价、止损/止盈位
- **自选池**：5 只宽基 ETF 监控（上证 50、沪深 300、创业板、科创 50、中证 1000）
- **评分系统**：0-5 分制，≥2 分触发买入信号
- **交易规则**：止损 -8%、止盈 +20%、市场环境过滤、仓位限制
- **定时分析**：每日自动分析并生成报告
- **绝对不自动下单**：仅用于分析和学习，不执行交易

## 触发条件
- 需要查询 ETF 实时行情时
- 需要进行 ETF 评分或分析时
- 需要查看持仓盈亏时
- 需要生成交易建议报告时
- 需要回测策略时

### 交易规则
- 买入阈值：≥2 分（0-5 分制）
- 止损：-8%
- 止盈：+20%
- 市场环境过滤：6 大宽基中 ≥4 只下跌 → 暂停买入
- 单一 ETF 仓位：≤ 总资金 50%
- **绝对不自动下单，不执行交易**

### 使用示例
```bash
# 运行每日分析
cd ~/.openclaw/workspace/stock-project
python daily_analysis.py

# 查看最新报告
ls -lt reports/ | head -1
cat reports/latest_report.md

# 查看持仓
cat data/portfolio.json | jq '.'
```
